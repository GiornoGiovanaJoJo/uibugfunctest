# Скриншоты ошибок (Bugs)

Несатированные скриншоты с найденными ошибками в результате UX-тестирования.

## Типы скриншотов:

- **Непонятные тексты** — bug-001
- **Необразимые элементы** — bug-002, bug-003
- **Перегруженные меню** — bug-005
- **Ошибки вывода** — bug-004

## Наименование файлов:

**Формат:** `bug-XXX-описание-YYYY-MM-DD.jpg`

**Примеры:**
- `bug-001-tokens-message-unclear-2025-12-11.jpg`
- `bug-003-payment-history-empty-2025-12-11.jpg`
- `bug-005-sidebar-menu-overwhelming-2025-12-11.jpg`

---

## Связь с Bug Report

Все эти скриншоты должны быть ссыланы в таблице **Bug Report** в колонке "Скриншоты / видео".

| Bug ID | Описание | Скриншот |
|-|-|-|
| BUG-001 | Tokens message не описан | `bug-001-tokens-message-unclear-2025-12-11.jpg` |
| BUG-003 | Payment history пуста | `bug-003-payment-history-empty-2025-12-11.jpg` |
| BUG-005 | Sidebar перегружен | `bug-005-sidebar-menu-overwhelming-2025-12-11.jpg` |
