# MAiGPT Testing Results

Структура папок для хранения результатов тестирования MAiGPT.

## Структура директорий

MAiGPT_Testing_Results/
  screenshots/           - Все скриншоты
  | - bugs/             - Скриншоты багов
  | - ui-issues/        - Скриншоты UX проблем
  | - performance/      - Проблемы производительности
  | - functional/       - Функциональное тестирование
  | - modules/          - По модулям MAiGPT
  | - desktop/          - Desktop версия
  | - mobile/           - Mobile версия
  | - chrome/           - Chrome браузер
  | - safari/           - Safari браузер
  videos/
  | - bug-reproduction/ - Видео воспроизведения багов
  | - user-flows/       - Видео пользовательских сценариев
  reports/
  | - bug-reports/      - Детальные отчёты о багах
  | - ux-reports/       - Отчёты об UX проблемах
  | - summaries/        - Итоговые резюме

## Правила именования файлов

Скриншоты: bug-001-description-2025-12-11.png
UX: ui-005-description-2025-12-11.png
Видео: bug-repro-001-description-2025-12-11.mp4

## Как добавлять скриншоты

1. Сделай скриншот (Win+Shift+S или Snipping Tool)
2. Назови по правилу: bug-001-register-button-2025-12-11.png
3. Положи в нужную папку: screenshots/bugs/
4. Обнови Excel с ссылкой на скриншот

## Инструменты для скриншотов

- Snipping Tool (встроенный в Windows)
- ShareX (бесплатный, расширенные возможности)
- Greenshot (лёгкий и простой)
